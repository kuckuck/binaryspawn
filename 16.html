LG 27GK750F-B Monitor Review

April 29, 2019


The main feature that differentiates this monitor is the high 240Hz refresh rate. This makes for smoother gameplay and a possible competitive edge over players using a lower refresh rate. Modern graphics cards are able to display over 240fps in current titles at 1080p with possibly some graphics settings lowered. The downside to this monitor is it uses TN panel technology which has worse colors and viewing angles compared to other panel technologies.

Specs are as follows:

Screen Size: 27"
Maximum Resolution: 1920 x 1080
Refresh Rate: 240 Hz
Viewing Angle: 170° (H) / 160° (V)
Brightness: 400 cd/m2 (typ) / 320 cd/m2 (min)
Response Time: 2ms (Faster), 1ms with Motion Blur Reduction
Panel: Twisted Nematic (TN)
Adaptive Sync Technology: FreeSync (AMD Adaptive Sync)
Connectors: 2 x HDMI v2.0, 1 x DisplayPort v1.2, USB3.0 (1up/2down) Support Quick Charge
